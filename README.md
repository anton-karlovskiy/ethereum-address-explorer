# UI

## Tech stack

- create-next-app
- TypeScript
- ESLint (`eslint-config-google`)
- `husky` & `lint-staged`
- `tailwindcss`
- `storybook`

## Getting the project up and running

```bash
yarn install
# development
yarn dev
# production
yarn build
yarn start
```
